# 羊了个羊 · React 迷你版

这是一个用 React + TypeScript 实现的双层匹配小游戏，使用 emoji 作为占位素材（后续可替换成图片）。

## 运行方式

1. 安装依赖：

```bash
npm install
```

2. 启动开发服务器：

```bash
npm run dev
```

3. 构建与预览：

```bash
npm run build
npm run preview
```

## 玩法

- 开局先选择难度：
	- 无脑：无限撤销、无限洗牌
	- 休闲：3 次撤销、1 次洗牌
	- 挑战：无撤销、无洗牌
- 点击最上层可见的方块，把它放进托盘。
- 托盘内出现 3 个相同图标会自动消除。
- 托盘最多 7 格，超出即失败。
- 支持撤销、洗牌、重开。

> 目前使用 emoji 作为占位素材，后续可以替换成你自己的图片资源。
